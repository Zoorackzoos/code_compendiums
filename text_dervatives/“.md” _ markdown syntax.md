# **“.md” / markdown syntax**

# Quick reference table

| Syntax | Output |
| ----- | ----- |
| `#` … `######` | Headings H1 … H6 |
| `*italic*` / `**bold**` | *italic* / **bold** |
| `` `code` `` | `code` |
| ```` ```code``` ```` | multi‑line code |
| `> quote` | \> quote |
| `- [x] task` | ✓ task |
| `[link](url)` | link |
| `![img](url)` | image |
| \` | a |
| `---` | horizontal rule |

# New line AKA “\\n” in this type of language

Just add two spaces after a sentence 

# Headings

\# H1  
\#\# H2  
\#\#\# H3  
\#\#\#\# H4  
\#\#\#\#\# H5  
\#\#\#\#\#\# H6

# Emphasis

\*italic\*      \_italic\_  
\*\*bold\*\*      \_\_bold\_\_  
\*\*\*bold+italic\*\*\*  
\~\~strikethrough\~\~

# Lists

\- Item A  
\* Item B  
\+ Item C

1\. First  
2\. Second  
   1\. Indented sub‑item  
3\. Third

# Task lists

\- \[ \] Incomplete task    
\- \[x\] Completed task

# Links & images

\#link  
\[Link text\](https://example.com)

\#image  
\!\[Alt text\](https://example.com/img.png "Title")

# Code 

**\#Inline code:**  
\<code\>`some_code()`\</code\>

\#“Fenced code block”  
\`\`\`python  
def foo():  
    return "bar"

# Block quotes

\> This is a quote    
\>\> Nested quote

\---  
\*\*\*
